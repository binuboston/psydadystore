# Dribbble Design System

This project uses a design system inspired by [Dribbble](https://dribbble.com/) — the community for graphic design.

## Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| **Primary** | `#ea4c89` | Dribbble hot pink — buttons, links, accents |
| **Foreground** | `#444444` | Primary text |
| **Muted** | `#9ba5a8` | Secondary text, icons |
| **Border** | `#eaeaea` | Borders, dividers |
| **Accent** | `#f5e0e8` | Light pink backgrounds, highlights |
| **Background** | `#ffffff` | Page background |
| **Muted BG** | `#f8f8f8` | Cards, inputs |

### Secondary Accent Colors (Dribbble palette)
- **Green**: `#8aba56`
- **Orange**: `#ff8833`
- **Cyan**: `#00b6e3`

## Typography

- **Font**: Inter (Google Fonts)
- **Weights**: 400 (normal), 500 (medium), 600 (semibold), 700 (bold)

## Usage

Use semantic Tailwind classes that map to the design tokens:

- `bg-primary` / `text-primary` — Dribbble pink
- `text-foreground` — Dark text (#444)
- `text-muted-foreground` — Gray text (#9ba5a8)
- `bg-muted` — Light gray backgrounds
- `bg-accent` — Light pink backgrounds
- `border-border` — Subtle borders
